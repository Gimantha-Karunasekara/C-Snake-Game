# C-Snake-Game

## Simple snake game made using C89

###Features:
* 	Food goal
* 	Load snake from file
*  *Unbeatable* feature (Won't loose the game)

###Steps to run:
1. run makefile
			
		$ make
		
1. Execute by providing snake file and food goal
	
		$ ./snake <filename> <food goal>
		
		Ex:
		$ ./snake snake.txt 10
